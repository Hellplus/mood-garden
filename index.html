About this garden

一个可以离线打开的情绪小花园

选择情绪、写下一句今天的心情，再种下一朵花。记录只保存在当前浏览器中，建议定期导出 JSON 备份。

查看使用说明
  • 选择一个情绪，写下一句心情，然后点击“种下一朵花”。
  • 使用筛选、搜索和排序整理历史花卡片。
  • “导出日记”会下载 txt 文本，“导出备份”会下载 JSON 数据。
  • “导入备份”可以合并或覆盖当前浏览器里的记录。
  • 所有记录保存在当前浏览器的 localStorage 中，清空花园会删除当前浏览器里的记录。

Welcome

欢迎来到 Mood Garden

这里是一页轻量情绪记录小花园。你可以从今天开始，慢慢种下自己的心情。

  1. 选择今天的情绪。
  2. 写下一句心情。
  3. 点击“种下一朵花”。
  4. 记录保存在当前浏览器，重要数据可以导出 JSON 备份。
