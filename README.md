# Mood Garden 情绪花园

Mood Garden 情绪花园是一个使用 HTML、CSS 和 JavaScript 从零完成的前端小作品。它把每天的心情记录成一张情绪花卡片，支持本地保存、随机花语、动态图标、筛选搜索、数据分析、主题切换和导入导出。

在线访问地址：[https://hellplus.github.io/mood-garden/](https://hellplus.github.io/mood-garden/)

## 项目简介

这是一个偏温柔、治愈风格的轻量情绪记录页面。用户可以选择今天的情绪，写下一句心情，然后种下一朵属于当天的花。每张花卡片会保存在浏览器 `localStorage` 中，刷新页面后仍然可以继续查看。

项目没有使用 React、Vue、构建工具或后端服务，主要用于练习原生前端开发中的页面结构、样式设计、DOM 操作、本地存储、数据渲染和 GitHub Pages 发布流程。

本项目只用于个人情绪记录和前端学习展示，不提供心理诊断或医疗建议。

## 主要功能

- 选择今日情绪：开心、平静、焦虑、疲惫、兴奋
- 写下一句今天的心情
- 点击“种下一朵花”生成情绪花卡片
- 根据情绪生成随机今日花语
- 每种情绪有独立图标池，种花时随机生成对应情绪图标
- 花卡片和 Hero 区图标带有轻量动画
- Hero 区文案会根据当前情绪动态变化
- 使用 `localStorage` 保存花卡片，刷新后自动恢复
- 支持编辑单条心情文字
- 支持删除单张花卡片，删除前二次确认
- 支持清空花园，清空前二次确认
- 支持按情绪筛选、按关键词搜索、最新优先 / 最早优先排序
- 显示当前筛选结果数量
- 支持导出 `.txt` 情绪日记
- 支持导出和导入 JSON 备份
- 导入备份前会预览数量和情绪分布
- 支持覆盖导入和合并导入
- 显示情绪统计、最近 7 天记录、情绪分布、最常出现情绪、连续记录天数和本周回顾
- 支持浅色、深色、治愈粉三种主题切换，并保存主题偏好
- 使用 Toast 轻提示反馈操作结果
- 支持空花园状态、无搜索结果状态和移动端响应式布局
- 手机端新增记录 / 花园 / 分析 / 数据四个区域
- 手机端使用底部导航降低长页面信息拥挤
- 高级功能收纳到数据区，桌面端保留完整展示布局

## 本地运行方式

这个项目可以直接在浏览器中运行：

1. 下载或克隆项目到本地。
2. 找到项目目录中的 `index.html`。
3. 双击 `index.html`，用浏览器打开即可。

也可以使用 VS Code 的 Live Server 插件打开，方便开发时自动刷新。

## 数据说明

- 所有花卡片数据都保存在当前浏览器的 `localStorage` 中。
- 刷新页面后，历史花卡片会从 `localStorage` 中恢复。
- 更换浏览器、清理浏览器数据或使用无痕模式，可能会看不到原来的记录。
- 点击“清空花园”会删除当前浏览器里的花卡片记录。
- “导出日记”会下载 `.txt` 文本文件，适合阅读和保存。
- “导出备份”会下载 JSON 数据文件，之后可以通过“导入备份”恢复或合并记录。

## 项目结构

```text
mood-garden/
├─ assets/
│  ├─ mood-garden-hero.png
│  ├─ mood-garden-cards.png
│  ├─ mood-garden-analytics.png
│  └─ mood-garden-theme.png
├─ index.html
├─ style.css
├─ script.js
├─ CHANGELOG.md
└─ README.md
```

文件说明：

- `index.html`：页面结构，包括 Hero、记录区、统计区、分析区、浏览区、花园列表和发布信息。
- `style.css`：页面样式，包括主题变量、响应式布局、花卡片、按钮、数据分析和动画效果。
- `script.js`：页面交互逻辑，包括种花、保存读取、筛选搜索、编辑删除、导入导出、主题切换和数据分析。
- `assets/`：项目截图，用于 README 展示。
- `CHANGELOG.md`：版本迭代记录。
- `README.md`：项目说明文档。

## 学习收获

通过这个项目，我主要练习了：

- 如何用 HTML、CSS、JavaScript 拆分一个完整页面
- 如何用 JavaScript 处理按钮点击、输入框和选择器
- 如何使用 `localStorage` 保存、读取和清空本地数据
- 如何用数组和对象组织情绪、花语、图标和文案配置
- 如何根据用户操作重新渲染页面
- 如何兼容旧数据字段，例如缺少 `id`、`flowerQuote` 或 `moodIcon`
- 如何实现筛选、搜索和排序，但不改变原始数据
- 如何生成 `.txt` 日记和 JSON 备份文件
- 如何读取用户上传的 JSON 文件并做基础格式校验
- 如何用 CSS 变量实现多主题切换
- 如何用 CSS `keyframes` 制作轻量动画，并支持 `prefers-reduced-motion`
- 如何做简单的数据统计和规则化回顾文案
- 如何整理代码中的常量配置、存储函数和统一刷新流程
- 如何根据屏幕尺寸调整页面信息架构
- 如何用移动端 tab 降低长页面复杂度
- 如何区分核心记录功能和导入导出等高级功能
- 如何把静态前端项目发布到 GitHub Pages
- 如何为项目准备 README、CHANGELOG 和上线前检查清单

## 注意事项

Mood Garden 情绪花园是一个轻量情绪记录工具，页面中的统计和回顾只基于用户自己写下的记录生成，不提供心理诊断、医疗建议或治疗服务。如果你正在经历持续的痛苦、焦虑或其他严重困扰，请及时联系可信任的人或专业支持。
