# Mood Garden 情绪花园

一个用 HTML、CSS 和 JavaScript 制作的小型前端练习项目。用户可以记录当天的情绪和一句心情，种下一张属于自己的情绪花卡片。

在线访问地址：[https://hellplus.github.io/mood-garden/](https://hellplus.github.io/mood-garden/)

## 项目简介

Mood Garden 情绪花园是一个偏温柔、治愈风格的私人情绪日记页面。页面已优化为治愈系情绪日记风格，支持浅色、深色、治愈粉三种主题、响应式布局和更完整的视觉体验。它不依赖后端，也不使用框架，数据保存在浏览器的 `localStorage` 中。

这个项目主要适合前端新手练习：

- HTML 页面结构
- CSS 布局和视觉样式
- JavaScript DOM 操作
- 浏览器本地存储 `localStorage`
- 简单的数据渲染和状态更新

## 主要功能

- 选择今日情绪：开心、平静、焦虑、疲惫、兴奋
- 输入一句今天的心情
- 点击“种下一朵花”生成情绪花卡片
- 根据不同情绪显示不同 emoji、颜色和文案
- 随机生成对应情绪的今日花语
- 花卡片保存到 `localStorage`
- 刷新页面后仍能看到历史花卡片
- 显示情绪统计小面板
- 支持按情绪筛选花卡片
- 支持按关键词搜索心情文字
- 支持最新优先 / 最早优先排序
- 支持一键重置筛选条件
- 显示当前筛选结果数量
- 支持导出情绪日记为 `.txt` 文本文件
- 支持导出 JSON 备份文件
- 支持导入 JSON 备份文件
- 导入前会预览记录数量和情绪分布
- 支持覆盖导入和合并导入
- 导入格式错误时会给出提示
- 支持编辑单条情绪记录的心情文字
- 支持删除单条情绪记录，删除前会二次确认
- 支持清空花园，清空前会二次确认
- 支持浅色 / 深色 / 治愈粉主题切换
- 支持保存主题偏好，刷新后保持上次选择
- 使用 Toast 轻提示展示操作反馈
- 区分空花园状态和筛选无结果状态
- 针对手机端做了简单的响应式体验优化

## 使用技术

- HTML
- CSS
- JavaScript
- localStorage

没有使用 React、Vue、构建工具或后端服务。

## 本地运行方式

这个项目可以直接在浏览器中运行：

1. 下载或克隆项目到本地。
2. 找到项目目录中的 `index.html`。
3. 双击 `index.html`，用浏览器打开即可。

也可以使用 VS Code 的 Live Server 插件打开，方便开发时自动刷新。

使用页面时，可以选择情绪、写下一句心情，然后点击“种下一朵花”保存记录。如果想保存一份文字版日记，可以点击“导出日记”按钮，下载包含历史花卡片的 `.txt` 文件。也可以点击“导出备份”保存 JSON 数据文件，之后通过“导入备份”恢复或合并记录。

## 项目文件结构

```text
mood-garden/
├─ index.html
├─ style.css
├─ script.js
└─ README.md
```

文件说明：

- `index.html`：页面结构，包括标题、情绪选择、输入框、统计面板和花园区域。
- `style.css`：页面样式，包括背景、按钮、卡片、统计面板和响应式布局。
- `script.js`：页面交互逻辑，包括种花、随机花语、保存数据、读取历史、情绪统计、筛选搜索、导入导出和清空花园。
- `README.md`：项目说明文档。
