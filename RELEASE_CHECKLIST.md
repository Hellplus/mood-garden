# Mood Garden 发布检查清单

每次发布或更新 GitHub Pages 前，可以按下面清单逐项检查。  
本清单面向 V2.9 最终产品体验验收，重点确认页面能公开展示，也能作为轻量本地记录工具长期使用。

## 页面访问

- [ ] GitHub Pages 地址可以正常打开。
- [ ] 页面没有明显加载错误、空白或控制台报错。
- [ ] 浏览器标题清楚。
- [ ] 页面版本显示为当前版本。
- [ ] 首页 Hero、Today 动态卡片、动态图标和动态文案正常。
- [ ] 首次使用引导可以正常显示和关闭。
- [ ] 页面明确说明数据保存在当前浏览器本地。
- [ ] 页面明确说明本项目不提供心理诊断或医疗建议。

## 每日记录路径

- [ ] 打开页面后能快速理解：选择情绪、写一句话、种下一朵花。
- [ ] 今日记录状态卡显示正确。
- [ ] 随机记录引导问题正常显示。
- [ ] “换一个”引导问题按钮正常。
- [ ] 可以选择情绪。
- [ ] 可以输入心情文字。
- [ ] 心情强度默认值合理。
- [ ] 今日关键词为空时也可以正常种花。
- [ ] 空心情文字不能提交，并有温柔提示。
- [ ] 种花成功后 Toast 和下一步引导正常。
- [ ] 最近 1-3 条记录快速回看正常。

## 花园管理

- [ ] 花卡片内容显示清楚：情绪、图标、日期、心情、花语、强度、标签、收藏状态。
- [ ] moodIcon 可以生成、保存并在刷新后保持不变。
- [ ] 图标专属动画正常，不影响阅读。
- [ ] 可以打开花卡片详情。
- [ ] 详情弹窗可以关闭、取消和保存。
- [ ] 详情中可以编辑心情文字、强度、标签、收藏状态和补充日记。
- [ ] 编辑为空时不会保存，并有提示。
- [ ] 收藏和取消收藏正常。
- [ ] 删除单朵花前有二次确认。
- [ ] 删除后列表、统计、分析、日历、标签云和最近记录同步更新。
- [ ] 清空花园前有二次确认。

## 浏览与筛选

- [ ] 按情绪筛选正常。
- [ ] 关键词搜索正常，并且输入时不会明显卡顿。
- [ ] 最新优先 / 最早优先排序正常。
- [ ] 只看收藏筛选正常。
- [ ] 标签云基于全部记录生成，数量正确。
- [ ] 点击标签可以筛选，再次重置后恢复全部。
- [ ] 强度筛选正常。
- [ ] 情绪、关键词、收藏、标签、强度和排序可以组合使用。
- [ ] 当前筛选状态提示清楚。
- [ ] 一键重置能清除所有筛选条件。
- [ ] 筛选只影响当前显示，不改变 localStorage 原始记录。

## 日历视图

- [ ] 列表 / 日历视图切换正常。
- [ ] 当前月日历显示正常。
- [ ] 上一月、下一月、回到今天正常。
- [ ] 今天有特殊标记。
- [ ] 有记录日期有视觉标记。
- [ ] 点击日期可以查看当天记录。
- [ ] 日历基于全部记录，不受当前筛选影响。
- [ ] 新增、删除、导入、清空后日历同步刷新。

## 数据分析与洞察

- [ ] 情绪统计总数正确。
- [ ] 每种情绪数量正确。
- [ ] 洞察总览显示正确。
- [ ] 今日 / 本周 / 本月切换正常。
- [ ] 最近 7 天统计正常。
- [ ] 最近 30 天趋势正常。
- [ ] 情绪分布正常。
- [ ] 最常出现情绪正常。
- [ ] 连续记录天数正常。
- [ ] 心情强度分析正常。
- [ ] 常用标签 Top 5 正常。
- [ ] 收藏记录回顾正常。
- [ ] 情绪 × 强度回顾正常。
- [ ] 数据不足时显示温柔空状态。
- [ ] 分析文案只做记录回顾，不做心理诊断或医疗建议。
- [ ] 分析基于全部记录，不受筛选、搜索、排序、标签、收藏或强度筛选影响。

## 导出与导入

- [ ] txt 情绪日记可以正常导出。
- [ ] txt 内容包含日期、情绪、心情、花语、强度、标签、收藏状态和补充日记。
- [ ] JSON 备份可以正常导出。
- [ ] JSON 备份包含 app、version、exportedAt 和 records。
- [ ] JSON 导入可以正常读取。
- [ ] 错误 JSON 文件会给出友好提示。
- [ ] 空 records 文件不会执行危险导入。
- [ ] 合并导入正常，并避免 id 冲突。
- [ ] 覆盖导入有二次确认。
- [ ] 导入后花园列表、日历、统计、分析、标签云、今日状态和最近记录同步刷新。
- [ ] 页面明确提醒长期保存建议导出 JSON 备份。

## 移动端体验

- [ ] 手机端默认进入“记录”区域。
- [ ] 底部导航包含记录 / 花园 / 分析 / 数据四个 tab。
- [ ] 当前 tab 状态清楚。
- [ ] 底部导航不遮挡主要内容。
- [ ] Toast 不遮挡关键按钮。
- [ ] 详情弹窗不被底部导航遮挡。
- [ ] 花园筛选区在手机端不横向溢出。
- [ ] 日历在手机端不横向溢出。
- [ ] 分析 tab 不出现严重信息堆叠。
- [ ] 表单控件和按钮容易点击。

## PWA 与缓存

- [ ] manifest.webmanifest 可以正常读取。
- [ ] service worker 可以注册。
- [ ] service worker 缓存版本与当前发布版本一致。
- [ ] 更新后不会持续读取旧缓存。
- [ ] 离线时可以打开基础页面。
- [ ] localStorage 数据在刷新后可以恢复。
- [ ] 直接双击 index.html 时不会因为 service worker 不可用而报错。

## 文档与公开展示

- [ ] README 包含线上访问地址。
- [ ] README 包含项目截图。
- [ ] README 包含主要功能、使用技术、本地运行方式和数据说明。
- [ ] README 当前版本与页面版本一致。
- [ ] CHANGELOG 记录到当前版本。
- [ ] PROJECT_BRIEF 真实、清楚、不过时。

- [ ] RELEASE_CHECKLIST 覆盖当前主要功能。
- [ ] 文档没有夸大为商业级产品。
- [ ] 文档明确说明不提供心理诊断或医疗建议。

- [ ] 项目中没有隐私数据。

- [ ] 项目中没有无效外部资源。
- [ ] 项目没有引入未说明的第三方库。
- [ ] GitHub Pages 使用的资源路径兼容 `/mood-garden/`。
- [ ] README 截图路径可以正常显示。
- [ ] 最终上线前完成桌面端和移动端各一次完整手动回归。
